# 종만북 (2022.04.09 ~ )

## 스터디 진행 방식

- 알고리즘 문제 해결 전략 책을 사용해서 진행한다
- 일주일에 정해진 분량만큼 책을 읽는다
- 스터디 시간에 각 챕터 읽은 내용을 설명한다
- 스터디가 끝나면 설명한 내용을 간략히 요약해서 깃에 올린다
